# Amina's Portfolio - Installation Guide

## 📦 Filer du har fått:

1. **index.html** - Din huvudsida (portfolio homepage)
2. **wemove-case-study.html** - Detaljerad WeMove case study
3. **style.css** - All styling för båda sidorna
4. **images/** - Mapp med alla bilder
5. **README.md** - Denna fil!

## 🚀 Hur du laddar upp till GitHub:

### Steg 1: Förbered filerna
Alla filer är redo att använda! Du behöver bara ladda upp dem.

### Steg 2: Ladda upp till GitHub

**Via GitHub.com (Enklast):**
1. Gå till ditt repository på GitHub
2. Klicka "Add file" → "Upload files"
3. Dra in ALLA filer (index.html, wemove-case-study.html, style.css)
4. Dra in hela `images/` mappen
5. Scroll ner och klicka "Commit changes"

**Via VS Code / Terminal:**
```bash
git add .
git commit -m "Add complete portfolio with WeMove case study"
git push
```

### Steg 3: Aktivera GitHub Pages
1. Gå till ditt repository Settings
2. Klicka "Pages" i vänster meny
3. Under "Branch", välj **main** (eller master)
4. Klicka **Save**

Din portfolio är nu live på: `https://DITT-ANVÄNDARNAMN.github.io/REPO-NAMN`

## ✏️ Hur du ändrar innehåll:

### Ändra text:
Öppna `index.html` eller `wemove-case-study.html` och sök efter texten du vill ändra.

### Ändra färger:
Öppna `style.css` och scrolla till toppen. Ändra värden under `:root {}`

### Lägg till fler projekt:
Kopiera `<article class="project-card">` sektionen i `index.html`

## 💡 Tips:

- Öppna `index.html` i webbläsare lokalt för att se hur det ser ut
- Använd VS Code för att redigera koden
- Testa på mobil efter du publicerat!
